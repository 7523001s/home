# ShutaSudo Portfolio

個人のポートフォリオサイトおよび講義関連のドキュメントです。

## ディレクトリ構成

- **講義** - 講義資料
  - `2025SIMD.html` - 2025年 SIMD講義
  - `2025SIMA.html` - 2025年 SIM-A講義
  - `過去の講義/` - アーカイブされた講義資料

- **Workshops** - ワークショップ資料
  - `PastWorkshop.html` - 過去のワークショップ

- **Diary** - 日記
  - `dairy2026.html` - 2026年日記

- **home** - ローカルホスト用ファイル

## ファイル

- `index.html` - メインページ
- `indexEng.html` - English version
- `watches.html` - 腕時計関連ページ
- `stylesheet.css` - スタイルシート
